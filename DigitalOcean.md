## Gradient AI SDK and ADK

Documentation and toolkits for building on DigitalOcean Gradient AI Platform.

- [Gradient SDK API Reference](https://gradient-sdk.digitalocean.com/api): Complete API reference for the Gradient AI Agentic Cloud SDK, covering Agents, Chat Completions, GPU Droplets, Knowledge Bases, Inference, and Models. Includes HTTP, TypeScript, and Python examples.
- [Gradient Python SDK Docs](https://gradientai-sdk.digitalocean.com/api/python): Python client library documentation for the Gradient AI Platform. Install via pip install gradient.
- [Gradient Agent Development Kit (ADK)](https://github.com/digitalocean/gradient-adk): Python toolkit for building, deploying, and operating production-grade AI agents on DigitalOcean. Framework-agnostic, supports LangGraph, LangChain, CrewAI, PydanticAI. Install via pip install gradient-adk.
